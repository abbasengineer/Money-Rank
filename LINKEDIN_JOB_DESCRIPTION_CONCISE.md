# MoneyRank - LinkedIn Job Description (Concise Version)

**Consumer Finance Decision Platform | 2025–Present**

Built MoneyRank, a consumer finance decision platform that gamifies financial education through daily, scenario-based challenges. Users rank financial options and receive immediate feedback on decision quality, consistency, and risk exposure.

**Key Technical Achievements:**

• **Scoring & Ranking System**: Implemented custom distance-based scoring algorithm (Kendall tau variant) with partial credit calculation, converting rankings into percentile-based social comparisons. Built real-time aggregate statistics engine with transactional updates for challenge retries.

• **Full-Stack Architecture**: Architected React + TypeScript frontend with Express.js backend, PostgreSQL database (Drizzle ORM), and RESTful API design. Implemented service-oriented architecture with 8+ specialized services (scoring, aggregates, streaks, badges, risk profiling).

• **User Engagement Systems**: Developed streak tracking, badge/achievement system (10+ badge types), retry wallet mechanics, and "best attempt" tracking. Built comprehensive analytics dashboard with score trends, category performance analysis, risk profiling algorithm, and Financial Health Score calculation.

• **Authentication & Payments**: Implemented multi-provider auth (anonymous, email/password, OAuth via Passport.js) with account linking. Integrated Stripe for subscription management (free/premium/pro tiers) with webhook handlers and customer portal.

• **Data & Performance**: Designed optimized PostgreSQL schema with strategic indexing. Built feature flags system, admin dashboard for content management, and forum/community features. Implemented SEO optimization and responsive design.

**Tech Stack**: React 19, TypeScript, Express.js, PostgreSQL (Supabase), Drizzle ORM, Stripe API, Passport.js, TanStack React Query, Framer Motion, Recharts, Tailwind CSS

Demonstrates ability to translate behavioral finance principles into measurable systems, architect full-stack applications with complex data relationships, and ship production-ready features from authentication to analytics—while maintaining code quality and user experience excellence.
